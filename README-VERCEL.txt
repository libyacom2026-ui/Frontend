Nomadic Energy - Vercel Ready

Environment Variables المطلوبة داخل Vercel:
SMTP_HOST
SMTP_PORT
SMTP_USER
SMTP_PASS
TO_EMAIL
FROM_EMAIL اختياري

مثال:
SMTP_HOST = mail.nomadicenergy.ly
SMTP_PORT = 465
SMTP_USER = info@nomadicenergy.ly
SMTP_PASS = كلمة مرور البريد
TO_EMAIL = info@nomadicenergy.ly
FROM_EMAIL = info@nomadicenergy.ly

ارفع محتويات هذا المجلد إلى GitHub ثم اربطه بـ Vercel.
